# Release v0.3.0.1 — Hotfix: Email Signature Tách File

**Ngày phát hành:** 2026-04-13
**Phiên bản:** 0.3.0.1 (từ 0.3.0)
**Nhánh:** main2.0

---

## 1. Tóm Tắt Phát Hành

- **Phiên bản:** 0.3.0 → 0.3.0.1
- **Mục tiêu phát hành:** Hotfix nhỏ — tách inline HTML chữ ký email khỏi code Python; loại trừ `signature_ceo.html` khỏi package build.
- **Phạm vi thay đổi:** `base/utils.py`, `pyproject.toml`, thêm 2 file template mới.

---

## 2. Thay Đổi Chính

### Sửa Lỗi
- **`TEMPLATE` trong `base/utils.py`**: Xóa ~230 dòng HTML chữ ký đang được nhúng cứng (hardcode) trong string Python. Thay bằng `{signature}` placeholder để tách biệt nội dung và logic.

### Thêm Mới
- **`_load_signature()`**: Helper function đọc `templates/admin/signature.html` tại thời điểm gọi `send_email_login`. Không cache — luôn đọc file mới nhất.
- **`templates/admin/signature.html`**: File HTML chữ ký CTO (Tu Anh Nguyen) — được build vào package, dùng cho email hệ thống.
- **`templates/admin/signature_ceo.html`**: File HTML chữ ký CEO (Hoa Nguyen) — chỉ dùng local, **không build vào package**.

### Cấu Hình
- **`pyproject.toml`**: Thêm `[tool.setuptools.exclude-package-data]` loại trừ `templates/admin/signature_ceo.html` khỏi wheel và sdist.

---

## 3. Files Thay Đổi

| File | Loại | Mô tả |

## 4. Tương Thích Và Rủi Ro

- **Tương thích ngược:** Hoàn toàn tương thích — không thay đổi API, model, migration.
- **Rủi ro:** Nếu `signature.html` bị xóa hoặc path sai, `send_email_login` sẽ raise `FileNotFoundError`. Cần đảm bảo file có mặt khi deploy.
- **Giảm thiểu:** File được đưa vào `package-data` → được đóng gói cùng wheel.
