# Release v0.3.4.5 — 2026-05-10

## Summary

Bugfix release improving visit tracking accuracy in `UserActivityMiddleware` and dashboard badge analytics. The middleware now correctly excludes HTTP redirects (301/302) from visit counts, and the `visitprofile_badge_callback` shows today's active visitors instead of all-time totals for more meaningful real-time metrics.

## Main Changes

### Added
- New migration `0027` altering `method` fields on `AnonymousActivity` and `UserActivity` models.
- Updated field constraints on `ExcelFile` and `PDFFile` in file management module.

### Fixed
- **Visit tracking accuracy**: `UserActivityMiddleware` now only tracks visits for non-redirect responses (`status_code not in (301, 302)`), preventing false visit counts from HTTP redirects triggered by authentication flows or URL canonicalization.

### Improved
- **Dashboard badge analytics**: `visitprofile_badge_callback` now filters VisitProfile records to today's date only (`last_seen__gte= midnight`), providing real-time active visitor count in the admin sidebar instead of cumulative all-time total.

### Removed
- Unused `user_badge_callback` function from `whiteneuron/base/utils.py`.

## Compatibility Notes

- **Schema change**: Migration `0027` alters existing fields on `AnonymousActivity` and `UserActivity` models. No data loss expected, but backup recommended before upgrading in production.
- **No breaking API changes**: All public interfaces remain unchanged.
- **Safe for all v0.3.x users**.

## Upgrade Guide

1. Install the new version:
   ```bash
   pip install --upgrade django-whiteneuron==0.3.4.5
   # or
   uv add "django-whiteneuron==0.3.4.5"
   ```

2. Apply database migrations:
   ```bash
   python manage.py migrate
   ```

3. Restart your application server (Daphne/Gunicorn).

## Rollback

Safe to revert to v0.3.4.4. If needed, rollback migrations with:
```bash
python manage.py migrate whiteneuron.base 0026
python manage.py migrate whiteneuron.file_management 0004
```

No schema changes or data loss expected during upgrade.
