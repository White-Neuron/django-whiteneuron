# v0.2.48.2 — 2026-04-10

## Security: Nâng cấp django 5.2.13 và thêm pyOpenSSL 26.0.0

### Tổng quan

Bản vá bảo mật bổ sung cho nhánh 0.2.48.x. Nâng cấp Django lên bản vá mới nhất (`5.2.13`) và thêm `pyopenssl==26.0.0` vào explicit dependencies để đảm bảo kiểm soát phiên bản TLS/SSL toàn bộ dependency tree.

---

### Changes

#### `pyproject.toml` — Dependency security update

- **Security**: `django` lower bound nâng từ `>=5.2.12` lên `>=5.2.13` — phiên bản vá bảo mật mới nhất của Django 5.2 LTS.
- **Security**: `pyopenssl==26.0.0` thêm vào `dependencies` dưới dạng explicit pin — tránh transitive dependency trôi nổi; `pyOpenSSL 26.0.0` vá các lỗ hổng trong TLS/SSL handling, certificate validation và cryptographic operations.

#### `uv.lock` — Lock file update

- Lock file được cập nhật tương ứng với `django 5.2.13` và `pyopenssl 26.0.0`.

---

### Files Changed

| File | Loại thay đổi |
|------|--------------|
| `pyproject.toml` | Security — nâng django bound, thêm pyopenssl pin |

### Compatibility Notes

- Tương thích hoàn toàn với Django 5.2.x LTS.
- Không có thay đổi API, template, hay migration — upgrade không rủi ro.
- Yêu cầu Python >= 3.11 (không đổi).

---

### Upgrade Guide

Cập nhật dependency trong dự án:

```bash
uv add "django-whiteneuron>=0.2.48.2"
uv sync
```

### Rollback

Nếu cần rollback về phiên bản trước:

```bash
uv add "django-whiteneuron==0.2.48.1"
uv sync
```
