# Release v0.2.32 — 2026-03-29

## 1. Tóm Tắt Phát Hành

- **Phiên bản:** 0.2.32
- **Mục tiêu phát hành:** Security hardening cho môi trường Cloudflare Tunnel + tối ưu middleware pipeline.
- **Phạm vi thay đổi:** `middleware.py`, `settings.py`, `env.example`, `README.md` — không có thay đổi model/migration.

## 2. Thay Đổi Chính

- **Security — IP spoofing fix:** `get_client_ip()` chỉ tin tưởng `CF-Connecting-IP` / `True-Client-IP` khi `BEHIND_CLOUDFLARE=True`. Trước đây bất kỳ client nào đều có thể gửi header giả để bypass rate limit.
- **Default BEHIND_CLOUDFLARE=True:** Vì luôn deploy qua Cloudflare Tunnel — mặc định an toàn và đúng với hạ tầng thực tế.
- **Performance — do_not_track cache:** `UserActivityMiddleware.__call__()` cache kết quả `do_not_track()` vào biến `skip`, tránh gọi 2 lần mỗi request (linear scan qua `exclude_paths`).
- **Cleanup:** Xóa `import logging` không sử dụng trong `middleware.py`.

## 3. Tương Thích Và Rủi Ro

- **Không có breaking change.** Các dự án không dùng Cloudflare cần set `BEHIND_CLOUDFLARE=False` trong `.env` (mặc định `True` từ version này).
- **Rollback:** Downgrade về `v0.2.31` không mất dữ liệu. Chỉ cần cập nhật `pyproject.toml`.
