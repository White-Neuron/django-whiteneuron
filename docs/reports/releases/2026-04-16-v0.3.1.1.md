# v0.3.1.1 — 2026-04-16

## Fix: `permission_viewer_callback` thiếu guard unauthenticated; thêm dependency `md2html-tailwind4`

### Overview

Bản vá nhỏ cho `v0.3.1` gồm 2 nội dung: (1) lỗi bảo mật nhẹ trong `permission_viewer_callback` — hàm trả về `True` ngay cả khi user chưa xác thực, (2) UNFOLD nav menu Feedbacks dùng sai permission callback, (3) thêm dependency mới `md2html-tailwind4`.

---

### Root Cause

| Vấn đề | Nguyên nhân |
|--------|-------------|
| `permission_viewer_callback` không guard unauthenticated | Hàm chỉ `return True` trực tiếp mà không kiểm tra `is_authenticated` — user ẩn danh (AnonymousUser) vẫn vượt qua |
| Feedbacks menu dùng sai callback | `settings.py` gán `permission_non_guest_callback` cho menu Feedbacks thay vì `permission_viewer_callback` như các menu khác |

---

### Changes

#### `whiteneuron/base/utils.py`

**Trước:**
```python
def permission_viewer_callback(request):
    return True
```

**Sau:**
```python
def permission_viewer_callback(request):
    if not request.user.is_authenticated:
        return False
    return True
```

- Guard rõ ràng cho AnonymousUser, tránh lộ menu item với user chưa đăng nhập.

#### `whiteneuron/base/settings.py`

**Trước:**
```python
"permission": "whiteneuron.base.utils.permission_non_guest_callback",
```

**Sau:**
```python
"permission": "whiteneuron.base.utils.permission_viewer_callback",
```

- Feedbacks menu item giờ dùng `permission_viewer_callback` — nhất quán với toàn bộ UNFOLD nav.

#### `pyproject.toml`

- **Added**: `md2html-tailwind4>=1.0.0` vào `dependencies`.
