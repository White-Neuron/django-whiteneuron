# Release v0.3.0.5 — 2026-04-14

## 1. Tóm Tắt Phát Hành
- **Phiên bản**: 0.3.0.4 → 0.3.0.5
- **Mục tiêu phát hành**: Sửa lỗi CSRF 403 trên production khi app chạy sau reverse proxy (nginx/Cloudflare).
- **Phạm vi thay đổi**: Nhỏ — chỉ thêm 1 dòng setting cho production.

## 2. Thay Đổi Chính
- **Fixed**: `whiteneuron/base/settings.py` — thêm `SECURE_PROXY_SSL_HEADER = ("HTTP_X_FORWARDED_PROTO", "https")` trong block `if not DEBUG`. Django giờ nhận biết request gốc là HTTPS khi chạy sau reverse proxy → CSRF origin check pass.

## 3. Tính Năng Mới
- Không có tính năng mới.

## 4. Tương Thích Và Rủi Ro
- **Đánh giá tương thích**: Hoàn toàn tương thích ngược. Setting chỉ active khi `DEBUG=False` và yêu cầu reverse proxy gửi header `X-Forwarded-Proto`.
- **Rủi ro còn lại**: Nếu reverse proxy không gửi `X-Forwarded-Proto`, setting không gây hại nhưng cũng không có tác dụng.
- **Kế hoạch giảm thiểu**: Đảm bảo nginx/Cloudflare config có `proxy_set_header X-Forwarded-Proto $scheme;`.

## 5. Xác Thực Đã Thực Hiện
- Build Tailwind: OK
- Makemigrations: No changes detected
- Build package (`uv build`): OK

## 6. Hướng Dẫn Nâng Cấp
- **Bước triển khai**: `uv pip install django-whiteneuron==0.3.0.5` — không cần migration hay thay đổi settings.
- **Điều kiện rollback**: Quay về v0.3.0.4 nếu có bất kỳ vấn đề nào.
