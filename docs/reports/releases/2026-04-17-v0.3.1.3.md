# Release v0.3.1.3 — 2026-04-17

## 1. Release Summary
- **Version**: 0.3.1.3
- **Release goal**: Bump `md2html-tailwind4` dependency to `>=1.1.0`; fix announcement modal URL resolution to use Django's `{% url %}` tag instead of hardcoded paths.
- **Scope**: Dependency update (1 package), template fix (1 file).

## 2. Main Changes

### Fixed
- **`templates/base/announcement.html`** — replaced hardcoded `/announcement/` path with `{% url 'announcement' %}` in both the `data-content-url` attribute and the JavaScript fallback URL. This ensures the URL is always resolved correctly regardless of URL prefix configuration.

### Updated
- **`pyproject.toml`** — bumped `md2html-tailwind4` minimum version from `>=1.0.0` to `>=1.1.0`; added `[tool.uv.sources]` entry pointing to the GitHub repo at rev `1.1.0` for reproducible installs.
- **`uv.lock`** — lock file refreshed to reflect updated dependency.
