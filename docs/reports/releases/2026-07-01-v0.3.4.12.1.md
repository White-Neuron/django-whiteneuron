# Release Notes: v0.3.4.12.1 (2026-07-01)

## Summary

Bugfix release addressing UserAdmin grid view field exclusions and list_display ordering for cleaner admin interface.

## Main Changes

### Fixed
- **UserAdmin.grid_exclude_fields_list_display**: Added `is_bot`, `is_active`, `is_staff`, `is_superuser`, `date_joined` to the grid view exclusion list, preventing unnecessary fields from appearing in card layout.
- **UserAdmin.list_display**: Reordered fields and removed `display_created`, `display_staff`, `display_superuser`, `uuid`; added `is_bot`, `is_staff`, `is_superuser`, `date_joined` for cleaner admin listing.
