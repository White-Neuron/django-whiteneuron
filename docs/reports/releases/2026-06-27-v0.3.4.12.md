# Release v0.3.4.12 — 2026-06-27

## Summary

**Release Type**: Cleanup & Admin UX Improvement
**Scope**: Repository hygiene, feedback system admin integration, dependency cleanup
**Risk Level**: Low — no schema changes, no breaking changes

---

## Main Changes

### Removed
- **`.DS_Store` files removed from git tracking** (7 files across repository)
  - Added `*/.DS_Store` to `.gitignore` for future prevention
  - Files removed: root, whiteneuron/, static/, templates/ directories

### Added
- **Feedback list in admin change form** (`FeedbackBaseAdmin`)
  - `render_change_form()` now loads `FeedbackData.objects.filter(content_type=ct, object_id=str(original.pk))` for the current model instance
  - Feedback entries passed to `change_form_outer_after_template` context as `feedback_list`
  - Requires template `admin/feedbacks/change_form_feedback_list.html` (new file)

### Removed (Dependencies)
- **`[tool.uv.sources]` block for `md2html-tailwind4`** removed from `pyproject.toml`
  - Dependency now resolved via PyPI — git source no longer needed

### Updated
- **Tailwind CSS regenerated** with v4.1.16 (previously built artifact)
