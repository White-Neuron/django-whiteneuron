### Tương Thích Và Rủi Ro

- **Không breaking change** trong source code — chỉ nâng version floor của dependencies.
- `django>=5.2.12` yêu cầu Python ≥ 3.10 (đã đáp ứng vì `requires-python = ">=3.11"`).
