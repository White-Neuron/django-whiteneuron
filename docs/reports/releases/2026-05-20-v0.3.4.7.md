# Release v0.3.4.7 (2026-05-20)

## Scope Summary

This release focuses on two areas: raising the Python minimum version requirement to 3.13, and fixing theme-aware rendering issues in the Markdown editor preview modal introduced in v0.3.3.

## New Changes

### Added/Improved
- **Python 3.13 requirement**: `requires-python` raised from `>=3.12` to `>=3.13`. This aligns with modern Python support and removes compatibility concerns for older Python versions.
- **Markdown editor modal theming fix**: The preview modal now uses explicit DaisyUI semantic tokens (`bg-base-100`, `text-base-content`) instead of relying on inherited styles, ensuring reliable rendering in both light and dark themes across all contexts.
- **Modal backdrop overlay**: Updated with explicit opacity classes (`bg-black/50` / `dark:bg-black/70`) for consistent appearance regardless of parent container styling.

### Fixed
- Removed unused import `UnfoldAdminCheckboxSelectMultiple` from `whiteneuron/base/admin.py`.

## Files Changed

| File | Type | Description |
|---|---|---|
| `pyproject.toml` | Config | Version bump, Python requirement, classifiers update |
| `whiteneuron/base/admin.py` | Cleanup | Removed unused import |
| `whiteneuron/static/base/forms/js/mdeditor.js` | Fix | Modal theme-aware styling improvements |
| `whiteneuron/static/base/css/styles.css` | Generated | Regenerated CSS with new utility classes |

## Compatibility Notes

- **Breaking**: Python 3.12 and earlier are no longer supported. Users on Python 3.12 must upgrade to 3.13 before installing this version.
- No database migrations required.
- No API changes.

## Validation Performed

- Tailwind CSS build: successful (280ms)
- Django migrations: no changes detected
- Package build (`uv build`): wheel and sdist generated successfully
- Version in `pyproject.toml`: 0.3.4.7

## Upgrade Guidance

1. Ensure your environment uses **Python 3.13 or later**.
2. Run `pip install --upgrade django-whiteneuron`.

## Rollback

Safe to revert to v0.3.4.6.3 by running:
```bash
pip install django-whiteneuron==0.3.4.6.3
```
Note: rollback requires Python 3.12 compatibility if downgrading from this version.
