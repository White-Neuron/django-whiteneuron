# Release v0.3.4.6.3 тАФ 2026-05-14

## Summary

Bugfix: Correct HTML/CSS/JS typo `action_buttoms` тЖТ `action_buttons` in grid view templates, restoring proper hover-reveal action buttons on admin object cards.

## Main Changes

### Fixed
- **Grid view templates**: CSS class `.action_buttoms` and JS selector `#action_buttoms_*` corrected to `.action_buttons` / `#action_buttons_*` across 2 template files тАФ restores the hidden/visible toggle of action buttons when hovering over grid view object cards.

| `pyproject.toml` | Version bump (0.3.4.6.2 тЖТ 0.3.4.6.3) |

## Compatibility Notes

- **No breaking changes** тАФ this is a pure typo correction.
- Safe for all v0.3.x users.

